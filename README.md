# File Browser - Chrome-Style File Browser

*hopefully......* A custom, high-performance file browser for Windows built using C++, Dear ImGui, and DirectX 11. 

## Project Motivation

I decided to make a file browser for my Windows laptop that behaves like a web browser. The goal is to create, fast, tabbed, and highly customizable *.....hopefully* . Also seemed like a great opportunity to learn.


## Tech Stack
* **Language:** C++
* **Graphics API:** DirectX 11
* **UI Library:** [Dear ImGui](https://github.com/ocornut/imgui)
* **OS Platform:** Windows (Win32 API)

## How to Build
To compile the project locally, open a Developer Command Prompt for Vs code, navigate to the root folder, and run:
shell.bat
build.bat
run.bat
